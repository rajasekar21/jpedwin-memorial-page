# Scalability Roadmap

## Phase 1: Static memorial

- Static Next.js export
- Local image archive
- Static biography, timeline, gallery, tributes, events
- GitHub Pages CI/CD

## Phase 2: Custom domain

- GoDaddy DNS
- GitHub Pages custom domain
- HTTPS enforcement
- WWW/non-WWW canonical redirect

## Phase 3: Dynamic platform

- Supabase PostgreSQL
- Memory and guestbook submissions
- Gallery upload workflow
- Admin moderation
- Row Level Security policies

## Phase 4: Advanced preservation

- Privacy-friendly analytics
- AI-assisted biography and memorial summaries
- QR code sharing for printed programs
- Printable tribute page
- Anniversary reminders
- RSVP for remembrance events
- Optional background music toggle with explicit user control
- Family tree visualization
